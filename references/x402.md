# x402 Pay-Per-Verify

## Overview

UZPROOF has built-in support for the **x402 HTTP payment protocol**. When enabled, AI agents can verify on-chain usage without API keys by paying per request with USDC on Solana.

**Current status:** x402 is implemented but not yet active on the production API. The API currently accepts requests freely. When x402 is enabled, the flow below will apply.

## How x402 Works (when enabled)

1. Agent sends verification request without API key
2. UZPROOF returns HTTP `402 Payment Required` with x402 payment details
3. Agent pays with USDC on Solana (automatic via x402 client)
4. Agent retries the request with `X-Payment` header — UZPROOF returns the result

## Integration (when x402 is enabled)

### With x402-proxy

Use [x402-proxy](https://github.com/cascade-protocol/x402-proxy) to auto-pay 402 responses:

```bash
npx x402-proxy
```

### Direct x402 Flow

```typescript
async function verifyWithX402(wallet: string, action: string, config: object) {
  // Step 1: Make request
  const res = await fetch('https://uzproof.com/api/verify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ wallet, taskType: action, config })
  });

  // Step 2: If 402, parse x402 payment requirements
  if (res.status === 402) {
    const paymentInfo = await res.json();
    // paymentInfo: { x402Version: 2, schemes: [{ scheme, network, maxAmountRequired, payTo, token, ... }] }

    // Step 3: Pay with USDC on Solana
    // ... execute USDC transfer to paymentInfo.schemes[0].payTo ...

    // Step 4: Retry with payment proof
    const retryRes = await fetch('https://uzproof.com/api/verify', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-Payment': paymentProof
      },
      body: JSON.stringify({ wallet, taskType: action, config })
    });

    return retryRes.json();
  }

  return res.json();
}
```

## Why x402 for Verification?

- **No API keys** — agents pay per use, no signup required
- **Micro-payments** — $0.05 per verify is viable for high-volume agents
- **Composable** — works with any x402-compatible agent (SP3ND, OpenDexter, QuickNode)
- **USDC on Solana** — stablecoin, near-zero gas, 400ms finality
