# x402 Pay-Per-Verify

## Overview

UZPROOF supports the **x402 HTTP payment protocol** — AI agents can verify on-chain usage without API keys by paying per request with USDC on Solana.

## How x402 Works

1. Agent sends verification request without API key
2. UZPROOF returns HTTP `402 Payment Required` with payment details
3. Agent pays with USDC on Solana (automatic via x402 client)
4. Agent retries the request — UZPROOF returns the result

## Integration

### With x402-proxy

The simplest approach — use [x402-proxy](https://github.com/cascade-protocol/x402-proxy) to auto-pay 402 responses:

```bash
npx x402-proxy
```

## Why x402 for Verification?

- **No API keys** — agents pay per use, no signup required
- **Micro-payments** — $0.05 per verify is viable for high-volume agents
- **Composable** — works with any x402-compatible agent (SP3ND, OpenDexter, QuickNode)
- **USDC on Solana** — stablecoin, near-zero gas, 400ms finality
